# Documentación de firma en Urus Capital

Este directorio contenía material de referencia de un proveedor externo de firma electrónica SaaS. **El producto usa firma simple in-house**: motor en Next.js, página pública `/firma/{token}`, OTP por SMS, PDF sellado y pista de auditoría.

**Fuente de verdad:** [docs/firma-digital.md](../firma-digital.md).

No mantener aquí credenciales ni tokens; usar solo `.env` y `.env.example`.
